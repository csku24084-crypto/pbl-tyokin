# pbl-tyokin

# PBL家計簿アプリ

## 🚀 機能の概要

・金額とカテゴリを入力することでその月に使用した金額を記録できることができる！
・カテゴリには「その他」も選択できて，自由に登録できる！
・自分で月の予算を設定できて，予算を超過すると警告してくれる！

## 📸 アプリの起動方法
- ## 起動方法（ローカル環境での実行手順）

このアプリケーションを自身のパソコンで起動する手順です。Pythonがインストールされている環境を前提としています。

### 1. 必要なライブラリのインストール
このアプリはWebフレームワークに `Flask` を使用しています。ターミナル（Windowsの場合はコマンドプロンプトやPowerShell、Macの場合はターミナル）を開き、以下のコマンドを実行してインストールしてください。

```bash
pip install Flask

プロジェクトが以下のディレクトリ（フォルダ）構成になっていることを確認してください。Flaskの仕様上、HTMLファイルは必ず templates という名前のフォルダ内に入れる必要があります。
pbl-tyokin/
├── app.py          # 提供されたPythonコード（ファイル名は任意、ここではapp.pyとします）
└── templates/
    └── index.html  # ポートフォリオ/WebアプリのHTMLファイル

ターミナルでプロジェクトのルートディレクトリ（app.py がある場所）に移動し、以下のコマンドを実行します。

python app.py

コマンドを実行すると自動的にブラウザが立ち上がり、アプリの画面（ http://127.0.0.1:5000 ）が開きます。
