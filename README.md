## 📸 アプリのこだわりポイント
- **「その他」入力欄の動的制御**: セレクトボックスで「その他」を選んだときだけ、JavaScriptで詳細入力欄がパッと表示されるようにしました。空欄のまま登録しようとすると必須エラーで安全に弾きます。
- **堅牢なエラーハンドリング**: サーバー側（Python）で全角数字を厳密に弾き、通信エラー（サーバー停止時）もフロント側（JavaScript）でキャッチして赤文字で安全に警告を出せるように設計しました。
