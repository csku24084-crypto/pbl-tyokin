# PBL家計簿アプリ (pbl-tyokin)

このアプリケーションは、ユーザーが入力した支出データをリアルタイムに計算し、設定した予算内での管理をサポートする簡易家計簿システムです。

---

## 🚀 主な機能

* **簡単支出記録**：金額とカテゴリを入力するだけで、その月の支出を手軽に記録。
* **柔軟なカテゴリ登録**：「その他」を選択することで、任意のカテゴリ名を自由に入力・登録可能。
* **安心の予算管理**：あらかじめ設定した月間予算（100,000円）を超過すると、画面に警告（アラート）を表示。
* **強固な入力チェック**：全角数字やマイナス、空欄などの不適切な入力を自動ではじくバリデーション機能を搭載。

💻 アプリの起動方法
ローカル環境（自身のパソコン）でアプリケーションを実行する手順です。
※パソコンに Python がインストールされている必要があります。

1. 必要なライブラリのインストール
このアプリはWebフレームワークに Flask を使用しています。
ターミナル（Windowsの場合はコマンドプロンプトやPowerShell、Macの場合はターミナル）を開き、以下のコマンドを実行してインストールしてください。

pip install Flask

2. ファイルの配置確認
プロジェクトが以下のディレクトリ（フォルダ）構成になっていることを確認してください。Flaskの仕様上、HTMLファイルは必ず templates という名前のフォルダ内に入れる必要があります。

pbl-tyokin/
├── app.py          # Pythonのバックエンドプログラム
└── templates/
└── index.html  # フロントエンド（画面）のHTMLファイル

3. アプリケーションの起動
ターミナルでプロジェクトのルートディレクトリ（app.py がある場所）に移動し、以下のコマンドを実行します。

python app.py

4. ブラウザでの確認
コマンドを実行すると、自動的にブラウザが立ち上がりアプリの画面（ http://127.0.0.1:5000 ）が開きます。

※自動で開かない場合は、ブラウザのアドレスバーに直接 http://127.0.0.1:5000 と入力してアクセスしてください。
